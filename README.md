# 🏠 House Price Prediction using Linear Regression

## 📌 Description

This project predicts house prices using Linear Regression in Google Colab. It includes data preprocessing, model training, prediction, and evaluation using MAE, MSE, and RMSE. The model analyzes relationships between features and house prices and is uploaded to GitHub for sharing and version control.

---

## 🚀 Features

* Data preprocessing and cleaning
* Train-test split
* Linear Regression model training
* Prediction of house prices
* Model evaluation with metrics
* Visualization of results

---

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

---

## 📊 Workflow

1. Load dataset
2. Preprocess data
3. Split data into training and testing sets
4. Train Linear Regression model
5. Predict house prices
6. Evaluate model performance

---

## 📈 Evaluation Metrics

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
